# 📱 Gerador de QR Code em Python

Aplicação simples em Python para gerar QR Codes de:

- Links
- Textos
- Wi-Fi
- Contatos (vCard)

---

## 🚀 Funcionalidades

- Gerar QR Code personalizado
- Gerar QR de redes Wi-Fi
- Gerar QR de contatos
- Escolher nome do arquivo
- Definir tamanho da borda

---

## ⚙️ Instalação

Instale a dependência:

```bash
pip install -r requirements.txt
```

---

## ▶️ Como usar

Execute o arquivo:

```bash
python main.py
```

Depois, siga as instruções no terminal.
